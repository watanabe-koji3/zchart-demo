# Zチャート・シミュレーター (経営分析ダッシュボード)

Zチャート（移動年計）を活用して、季節変動を除いた「企業の真の業績トレンド」を可視化・シミュレーションするアプリです。

## 概要

売上には通常、「繁忙期」や「閑散期」といった季節ごとの波があります。このツールでは、そうした季節性を取り除いた**「移動年計（Moving Annual Total）」**を自動計算し、以下の3本の線で会社の成長性を診断します。

1.  **🔴移動年計**: 過去1年間の実績合計（実力トレンド）
2.  **🟢売上累計**: 当期の積み上げ実績
3.  **🔵月次売上**: 毎月の売上（季節変動を含む）

## 使い方

1.  **基礎数値を設定する**:
    左側のサイドバーで、「期首（スタート時）の売上」と「期末（ゴール時）の目標売上」を設定してください。右肩上がりの計画か、現状維持かを確認できます。
2.  **季節変動を調整する**:
    「季節指数」を変更して、あなたのビジネスの繁忙期（1.2など）や閑散期（0.8など）を再現してください。
3.  **グラフを確認する**:
    設定に応じて「Zチャート」がリアルタイムに描画されます。
4.  **詳細データの確認**:
    画面下部では、詳細なデータテーブル（CSVダウンロード可能）や、印刷用のグラフ（Matplotlib製）も確認できます。

## 機能一覧

* **インタラクティブなZチャート**: Plotlyを使用した、カーソルを合わせると数値が見える動的なグラフ。
* **季節指数のカスタマイズ**: 月ごとの売上の波を自由に設定可能。
* **学習コンテンツ**: 「なぜ2年分のデータが必要なのか？」など、Zチャートの仕組みを図解で解説。
* **印刷用グラフ**: 資料作成に使える静止画グラフの生成。
* **データ出力**: シミュレーション結果のCSVダウンロード。
